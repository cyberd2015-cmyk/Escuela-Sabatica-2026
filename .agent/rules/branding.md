# Institutional Branding: "Sello 1853"

## Design Principles
All new components and modifications must adhere to the 1853 Institutional Seal standards.

## Component Rules
- **Borders**: ANY new component must use `border-institutional-gold`.
- **Rounding**: EVERY new component must use `rounded-xl`.

## Typography Rules
- **Headers**: Heading typography must be `uppercase` and have `tracking-widest` to replicate the institutional seal look.

## Color Palette
- **Institutional Blue**: `#1A4789` (Primary Background)
- **Institutional Gold**: `#D4AF37` (Accent/Borders)
- **Institutional White**: `#FDFDFD` (Layout/Contrast)

## Gradients
- **Hero Gradient**: Always use `bg-hero-gradient` for main hero sections or significant backgrounds.
